# E-Nutrition API

Backend services for nutrition planning and health-tracking scenarios.

## What this repository contains
- .NET backend code for nutrition domain workflows
- Tooling/scripts for local development support

## Tech stack
- C# / .NET

## Project structure
- `src/` - application source code
- `tools/` - development and support scripts

## Getting started
1. Install a compatible .NET SDK.
2. Restore packages from the solution/project in `src/`.
3. Build and run the API locally.

## Notes
Adjust local configuration files for your environment before running in production-like setups.
